# GTG Perfumes Landing Page

This repository contains the source code for the GTG Perfumes landing page, developed as part of a front-end web development assignment. 

## 🚀 Objective
The goal of this project was to convert a Figma design into a fully responsive, pixel-perfect webpage using purely **HTML, CSS, and JavaScript** without the use of frameworks like React, Vue, or TailwindCSS. The project demonstrates front-end fundamentals, attention to detail, and the ability to produce clean, maintainable code.

## ✨ Features Implemented
- **Pixel-Perfect UI**: The webpage structure, typography, and spacing meticulously match the provided Figma designs.
- **Fully Responsive Design**: Fluid layout adapting seamlessly to Mobile, Tablet, and Desktop screens.
- **Interactive Mobile Navigation**: A functional hamburger menu for smaller screens.
- **Dynamic Product Gallery**: A fully interactive image gallery allowing users to navigate via left/right arrows, pagination dots, or thumbnail selection.
- **Dynamic "Add to Cart" Logic**: JavaScript functionality that calculates 9 different variations based on selected radio options (Fragrances and Purchase Type) and dynamically updates the cart submission link.
- **Expandable Subscription Options**: UI elements for Single & Double Subscription tiers that expand and collapse cleanly upon selection.
- **Animated Data Counters**: Smooth counting animations from 0 to target percentages (e.g., 84%, 78%) when scrolling into view.
- **HTML Comparison Table**: A semantic, visually styled HTML table outlining why GTG is the #1 choice.
- **Performance Optimized**: Minimized resources, clean standard CSS, and lazy-loading implemented for smooth performance.

## 🛠️ Technologies Used
- HTML5 (Semantic and Accessible structure)
- CSS3 (Custom styling, Flexbox/Grid layouts, Media Queries, CSS Transitions/Animations)
- Vanilla JavaScript (DOM Manipulation, Event Listeners)
